How to Use
1
Set up your cookies Export cookies from openrent.co.uk using the Cookie-Editor browser extension and paste them in the box on the right. This lets the dashboard submit enquiry forms on your behalf.
2
Go to Calling → select a number Pick which Twilio number to call from. This should match the number associated with the account you want enquiries sent from.
3
Optionally scrape new listings Use "Scrape Rightmove Now" to pull the latest property references into the queue. Only new ones (not already called) will be added.
4
Run a batch of calls Set a batch size and hit "Run Next Batch". The system calls Rightmove's automated line and enters each reference number via the keypad. Watch the live log for progress.
5
Check SMS & Enquiries Successful calls trigger an SMS back from Rightmove containing a link to submit an enquiry on OpenRent. Go to the SMS & Enquiries tab and click "Submit All Pending" to send them.
6
Reply to landlords Once an enquiry is submitted, landlords may message back on OpenRent. Go to the Replies tab, paste your thread URL (or load unread threads), and the AI will draft a reply to get their phone number.
OpenRent Session Cookies

Export your openrent.co.uk cookies using the Cookie-Editor browser extension (Export → JSON), then paste here. Cookies are needed to submit enquiry forms.

Steps to export cookies:

  1. Install Cookie-Editor from the Chrome/Firefox extension store
  2. Log in to openrent.co.uk in your browser
  3. Click the Cookie-Editor icon → click Export → Export as JSON
  4. Paste the copied JSON into the box above and click Save
